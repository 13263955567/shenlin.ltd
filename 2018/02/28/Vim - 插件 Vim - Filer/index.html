Vim 插件 VimFiler

发表于 2018-02-28 | 分类于 Vim ， VimFiler

VimFiler

配置项

定义vimfiler位置 topleft or rightbelow

自动切换到编辑文件的目录

快捷键

TAB         切换窗口
S           排序，排序选项里有 save 功能，可以保存排序选项
t           展开目录
T           递归展开目录
<S-SPC>     标记文件或目录
N            新建文件
r           重命名
q            隐藏
L            驱动器列表
~            切换到家目录
\            切换到根目录（/ 还是vim查找）
