--with-apxs2就不会生成libphp5.so


--prefix= \
--with-config-file-path= \

--with-apxs2= \

--with-mysql=/usr/bin/mysql_config  \

--with-mysqli=mysqlnd \
--with-mysqli=/usr/local/mysql/bin/mysql_config \

--with-pdo-mysql=mysqlnd \
--with-pdo-mysql=/usr/local/mysql \






  --with-apxs2=FILE       Build shared Apache 2.0 Handler module. FILE is the optional
                          pathname to the Apache apxs tool apxs

  --enable-mysqlnd        Enable mysqlnd explicitly, will be done implicitly
                          when required by other extensions

  --with-mysqli=FILE      Include MySQLi support.  
			  FILE is the path to mysql_config.
			  If no value or mysqlnd is passed as FILE, the MySQL native driver will be used


  --with-pdo-mysql=DIR    PDO: MySQL support. 
			  DIR is the MySQL base directory
                          If no value or mysqlnd is passed as DIR, the MySQL native driver will be used




如果之前编译(make)失败,记得用make clean 或者 make distclean 清除之前编译的缓存文件,然后再重新make && make install

--enable-mysqlnd

--with-pdo-mysql=mysqlnd
