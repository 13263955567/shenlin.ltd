MySQL `DROP FUNCTION` 语法

<!--more-->

`DROP FUNCTION` 语句用于删除存储函数和用户定义函数(UDFs):

* 删除存储函数，参考 Section 13.1.26, “DROP PROCEDURE and DROP FUNCTION Syntax”.

* 删除用户定义函数，参考 Section 13.7.4.2, “DROP FUNCTION Syntax”.
